# 网页 UI 动效灵感库 | Web Motion Showcase

[简体中文](./README.md) | [English](#english)

本项目是一个专为网页设计初学者量身打造的互动式动效科普与灵感库。

## 🌟 项目特性 | Features
- **克制双色温主题 (Calm Multi-Theme)**：精心校准的 Slate (石板灰)、Sage Green (莫兰迪绿) 与 Warm Sand (秋叶暖沙) 柔和主题，远离刺眼纯白。
- **30+ 动效全屏预览 (Fullscreen Previews)**：收录常用的网页 UI 动效，支持交互参数调整与一键复制 AI 提示词。
- **性能调优 (Performance Optimized)**：对跑马灯与视差滚动采用 1:1 位移转化，并重构了网格重排 (Layout Transition) 折叠物理动效。

## <a name="english"></a>English Description

An interactive educational catalog and inspiration library of web UI motions tailored for beginner designers and developers.

- **Soothing Theme System**: Eye-friendly, low-contrast Slate, Sage Green, and Warm Sand color palettes (supporting Light & Dark modes) that completely avoid pure white glare.
- **30 Classic UI Motions**: Interactive fullscreen sandbox environment with instant clipboard copies for AI generator prompts.
- **Dynamic Physics & Transitions**: Smoothed 1:1 scroll translations for marquees and parallax pages, and fluid folding animations for grid-reordering filters.
