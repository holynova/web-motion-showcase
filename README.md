# 网页 UI 动效灵感库 (Web Motion Showcase)

本项目是一个为网页设计初学者量身打造的互动式动效科普与灵感库。

## 🌟 项目特性
- **克制美学**：精心校准的 Slate、Sage Green、Warm Sand 双色温低饱和主题。
- **全屏沉浸体验**：收录 30 个经典 UI 动效，支持参数微调与一键复制 AI 开发提示词。
- **流畅跑马灯与视差**：对滚动和重排过滤动画进行了 1:1 响应性能优化。
